# Sales Dashboard

A beginner-friendly analytics project that cleans sales data, computes KPIs, and visualizes trends in Streamlit.

## Tech Stack

- Python
- Pandas
- SQL (query examples)
- Streamlit
- Plotly

## Features

- Revenue, order count, and average order value KPIs
- Monthly revenue trend chart
- Region-wise and category-wise breakdown
- Data cleaning step before analytics

## Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Files

- `app.py`: Streamlit dashboard
- `src/clean_data.py`: cleaning and feature engineering logic
- `data/sales_sample.csv`: sample dataset
- `sql/analysis_queries.sql`: SQL aggregation examples
